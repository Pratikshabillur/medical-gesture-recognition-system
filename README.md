# 🏥 Medical Gesture-Based Patient Communication System

## 📌 Overview
This project is an AI-powered system that enables non-verbal or physically challenged patients to communicate with medical staff using body gestures. The system uses computer vision and deep learning to recognize gestures and convert them into medical text and speech output.

---

## 🛠️ Technologies Used
- Python  
- TensorFlow  
- OpenCV  
- Streamlit  
- MoveNet Pose Estimation  
- Bi-LSTM Neural Network  

---

## ⚙️ Features
- Real-time gesture recognition using webcam  
- Pose estimation using MoveNet model  
- Gesture classification using Bi-LSTM  
- Converts gestures into medical text  
- Speech output for accessibility  
- Runs efficiently on CPU  

---

## 🧠 Model Architecture
- MoveNet for extracting human body keypoints  
- Bi-LSTM for gesture sequence classification  
- Custom dataset trained for medical gestures  

---

## 📊 Results
- Achieved **92.8% accuracy**  
- Low-latency real-time predictions  

---

## 🚀 Use Cases
- Hospitals and healthcare centers  
- Non-verbal or paralyzed patients  
- Assistive healthcare communication  
- Emergency gesture-based communication
